## 🚀 Projects

### 🏥 Healthcare Data Warehouse
Production-style data warehouse built on the Synthea 
synthetic patient dataset using PostgreSQL.
- Three-layer pipeline: raw → staging → warehouse
- SCD Type 2 patient dimension with full history tracking
- Fact tables with point-in-time joins
- Advanced SQL: window functions, CTEs, cohort analysis, 
  query optimisation

### 👥 Customer Segmentation — K-Means Clustering
Customer segmentation model on e-commerce/retail data.
- K-Means clustering on purchasing behaviour patterns
- Elbow method and silhouette analysis for optimal clusters
- Actionable segment profiling and business recommendations

### 🤖 AI RAG Document Q&A Bot
Retrieval-Augmented Generation system for Q&A over documents.
- Document ingestion pipeline with chunking and embeddings
- FAISS vector store for similarity search
- Llama LLM for grounded, context-aware responses
